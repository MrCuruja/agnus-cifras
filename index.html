Configurar o Firebase

Para o Ágnus Cifras salvar e sincronizar as cifras automaticamente com toda a banda, ele precisa de um banco de dados Firestore gratuito do Firebase.

  1. Acesse console.firebase.google.com e crie um projeto.
  2. No menu lateral, abra Firestore Database → Criar banco de dados (pode iniciar em modo de teste).
  3. Vá em Configurações do projeto → role até "Seus apps" → clique no ícone </> Web para registrar um app.
  4. Copie o objeto firebaseConfig mostrado.
  5. Abra este arquivo .html em um editor de texto, procure por SUA_API_KEY perto do topo do código e cole os seus valores no lugar.
  6. Salve o arquivo e abra novamente no navegador.

⚠️ Depois de configurar, ative também a Autenticação (Email/Senha) e as Regras do Firestore — veja as instruções que te passei junto com este arquivo.
